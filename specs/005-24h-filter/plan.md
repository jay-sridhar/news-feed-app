## Summary

Filter the live news feed to only show articles published within the last 24 hours. A pure utility function (`isRecent`) checks each article's `pubDate`; articles with missing or unparseable dates are always shown. The filter is applied in `useFeed.ts` after fetching so both the paginated list and the search index stay fresh. A new "No recent articles" message replaces the retry UI when a successful fetch yields zero displayable articles.

## Implementation Notes

### `src/utils/articleAge.ts` (new)

```ts
export const FRESHNESS_WINDOW_MS = 24 * 60 * 60 * 1000

export function isRecent(pubDate: string): boolean {
  if (!pubDate || pubDate.trim() === '') return true
  const date = new Date(pubDate)
  if (isNaN(date.getTime())) return true
  return Date.now() - date.getTime() <= FRESHNESS_WINDOW_MS
}
```
